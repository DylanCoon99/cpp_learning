# C++ Learning Plan

## Phase 1 — C++ Foundations
1. `01_hello_cpp` — Compilation, iostream, namespaces
2. `02_types_and_variables` — Built-in types, auto, const, type conversions
3. `03_control_flow` — if/else, loops, switch, range-based for
4. `04_functions` — Overloading, default args, pass-by-value/reference
5. `05_references_and_pointers` — References vs pointers, nullptr, new/delete
6. `06_strings_and_arrays` — std::string, std::array, std::vector
7. `07_structs_and_enums` — Structs with methods, scoped enums, aggregate init
- **P1_project** — Contact Book CLI application

## Phase 2 — Object-Oriented C++
8. `08_classes` — Constructors, destructors, access specifiers, RAII
9. `09_operator_overloading` — Arithmetic, comparison, stream operators
10. `10_inheritance` — Virtual functions, polymorphism, abstract classes
11. `11_rule_of_five` — Copy/move constructors, copy/move assignment, destructor
- **P2_project** — TBD

## Phase 3 — Modern C++ & the Standard Library
12. `12_smart_pointers` — unique_ptr, shared_ptr, weak_ptr
13. `13_templates` — Function/class templates, template specialization
14. `14_stl_containers` — map, set, deque, unordered variants
15. `15_iterators_and_algorithms` — STL algorithms, custom iterators
16. `16_lambdas` — Captures, std::function, functional patterns
- **P3_project** — TBD

## Phase 4 — Intermediate Topics
17. `17_error_handling` — Exceptions, noexcept, error codes vs exceptions
18. `18_move_semantics_deep` — Rvalue refs, perfect forwarding, std::move
19. `19_concurrency` — std::thread, mutexes, atomics, async/future
20. `20_file_io` — Streams, binary I/O, filesystem library
- **P4_project** — TBD

## Phase 5 — Advanced
21. `21_template_metaprogramming` — SFINAE, constexpr if, type traits
22. `22_design_patterns` — Factory, observer, strategy in idiomatic C++
23. `23_memory_model` — Allocators, placement new, alignment
24. `24_build_systems` — CMake, linking, header/source organization
25. `25_capstone_project` — A substantial project pulling it all together
